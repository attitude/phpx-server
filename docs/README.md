# phpx-server docs

This site is built with [Docusaurus](https://docusaurus.io/). It's deployed
to GitHub Pages by `.github/workflows/docs.yml` on every push to `main`.

## Installation

```bash
pnpm install
```

## Local development

```bash
pnpm start
```

This starts a local dev server and opens a browser window. Most changes are
reflected live without restarting the server.

## Build

```bash
pnpm build
```

This generates static content into the `build` directory.
